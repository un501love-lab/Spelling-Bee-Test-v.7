Spelling Bee Test

ภาคเรียน:
คลังคำศัพท์ของแต่ละเทอมแยกเก็บจากกัน

เพิ่มคำศัพท์ ไม่จำกัดจำนวน

เพิ่มหลายคำพร้อมกัน (วางเป็นบรรทัด หรือคัดลอกเซลล์จาก Excel / Google Sheets มาวาง)

พิมพ์บรรทัดละ 1 คำ รูปแบบ คำ | ประโยค เช่น apple | I eat an apple every day.
หรือคัดลอก 2 คอลัมน์ (คำ, ประโยค) จาก Excel / Google Sheets มาวางในช่องนี้ได้โดยตรง

นำเข้า / ส่งออกไฟล์ Excel · Google Sheets · CSV

อัปโหลดไฟล์ Excel (.xlsx) หรือ CSV — คอลัมน์ A = คำศัพท์, คอลัมน์ B = ประโยคตัวอย่าง, คอลัมน์ C = ระดับชั้น เช่น ป.1 (มีหรือไม่ก็ได้)
📑 ไฟล์ Excel ที่แยกชีตเป็นระดับชั้น (เช่น ชีตชื่อ ป.1, ป.2, … ป.6) ระบบจะอ่านทุกชีตและติดป้ายระดับชั้นให้ตามชื่อชีตอัตโนมัติ
จาก Google Sheets: เมนู ไฟล์ → ดาวน์โหลด → Microsoft Excel (.xlsx) แล้วนำมาอัปโหลดที่นี่ (ได้ทุกชีตพร้อมกัน) หรือคัดลอกเซลล์มาวางในช่อง "เพิ่มหลายคำพร้อมกัน" ด้านบนก็ได้
คำที่นำเข้าจะเข้าสู่คลังของเทอมที่เลือกอยู่ด้านบน
🌐 คลังกลางสำหรับลิงก์เว็บไซต์ (GitHub): อัปโหลดไฟล์คลังคำไว้ที่เดียวกับ index.html บน GitHub ตั้งชื่อว่า words-term1.xlsx / words-term2.xlsx (ไฟล์ Excel แยกชีต ป.1–ป.6 ได้) หรือ words-term1.csv / words-term2.csv (กด "ดาวน์โหลดคลังเป็น CSV" ได้ไฟล์ชื่อถูกต้องพร้อมใช้) — ทุกคนที่เปิดลิงก์จะได้คลังคำชุดนี้อัตโนมัติ อัปโหลดไฟล์ใหม่ทับได้เรื่อย ๆ คนอื่นรีเฟรชหน้าก็ได้เวอร์ชันล่าสุด

คำศัพท์ทั้งหมด 0 คำ

    ยังไม่มีคำศัพท์ — เพิ่มคำแรกด้านบน หรือ

    เลือกคำที่จะสอบ สูงสุด 30 คำ

    กดปุ่มระดับชั้นด้านล่างเพื่อกรองก่อน หมายเลขจะเริ่มนับ 1 ใหม่ในชั้นนั้น แล้วพิมพ์เลือกแบบช่วงหรือเจาะจง เช่น 1-10 หรือ 2,5,8-12 หรือพิมพ์คำศัพท์เอง เช่น apple, tiger แล้วกด "เลือกตามที่พิมพ์" — หรือคลิกติ๊กเลือกจากรายการก็ได้ (เลือกข้ามหลายชั้นได้ สลับตัวกรองแล้วเลือกเพิ่มได้เลย)

    เลือกแล้ว 0 / 30 คำ

      คลังคำศัพท์ยังว่าง — ไปเพิ่มคำที่แท็บ "คลังคำศัพท์" ก่อน

      แชร์ชุดข้อสอบนี้ ลิงก์

      ลิงก์จะฝังคำศัพท์และประโยคที่เลือกไว้ในตัว ผู้รับเปิดแล้วเริ่มสอบได้ทันที ไม่ต้องพิมพ์เอง

      เลือกแล้ว 0 / 30 คำ

      ตั้งค่าเสียงและเวลา American English

      ⚠️ เบราว์เซอร์นี้ไม่มีเสียงอ่านภาษาอังกฤษ — แนะนำให้เปิดลิงก์นี้ใน Google Chrome หรือ Microsoft Edge บนคอมพิวเตอร์ / มือถือ แล้วลองใหม่

      ลำดับการอ่านแต่ละข้อ: "Number 1." → อ่านคำตามจำนวนรอบที่ตั้ง (เว้นจังหวะ) → อ่านประโยค → เวลาเขียน → เสียงกริ่ง 🔔 → ข้อถัดไป
